# Кейс 4. Разработка системы матчинга и дедупликации профилей клиентов на маркетплейсе скидок

## Цель
Цель: создать алгоритм и прототип системы автоматического поиска и объединения дубликатов профилей 
клиентов по неполным и зашумленным данным (ФИО, email, телефон, дата рождения, город и др.), 
чтобы повысить качество клиентской базы, точность персонализации и корректность аналитики на маркетплейсе.

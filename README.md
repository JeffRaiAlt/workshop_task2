# Кейс 2. Прогнозирование сбоев и рекомендации диспетчеру на основе данных группы промышленных датчиков

## Цель

Разработать MVP-решение, которое анализирует данные группы связанных промышленных датчиков, прогнозирует ухудшение 
состояния системы и возможные сбои, выявляет аномалии как ранние сигналы и формирует конкретные рекомендации диспетчеру по реагированию.
